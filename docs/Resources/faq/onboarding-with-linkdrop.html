Onboarding With LinkDrop

An easy way to onboard someone new to the NEAR ecosystem is by sending them a linkdrop URL to which will send the receiver to the NEAR wallet as shown below.

Warning: Links generated are only stored in local storage right now. If you reset local storage you will lose the links, so make sure to download all links into CSV as a backup as shown in section 10.
